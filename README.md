# 二轮差速运动模型的ROS驱动

---

## 引言

本项目为完成一个轮椅的ROS驱动，以在该轮椅上实现导航的功能。
这也是我第一次在项目开发中使用git，我希望通过该项目，提高我对于git，以及Python、Markdown等工具的使用能力，同时有意识的提高代码的规范性，当然我也会再次强调，我觉得代码最核心的内容是代码本身，代码思想本身的简洁是最高的追求，此外，项目开发的效率也是放在首位的，技术只是服务。

## 计划
